# Terence Musa Dlamini — Portfolio

A modern, responsive, and visually polished portfolio website showcasing my journey as a **Cybersecurity & Cloud Infrastructure Specialist**. Built with **Next.js**, **TailwindCSS**, and **shadcn UI**, the site blends professional case studies, technical experiments, certifications, and personal interests into one cohesive digital experience.

---

## ✨ Features

- **Modern Retro Glass UI**  
  Aurora gradients, neon grid overlays, and subtle glassmorphism for a futuristic yet professional look.

- **Single‑Page Scroll Layout**  
  Smooth anchor navigation with parallax and hover animations.

- **Sections**
  - 🏠 **Home** — Hero portrait, headline, and dock‑style navigation.  
  - 📂 **Projects / Labs** — Case studies in Cloud Infrastructure, SIEM dashboards, and automation scripts.  
  - 🧪 **Experiments** — Creative prototypes and TryHackMe progress.  
  - ✍️ **Writing / Notes** — Embedded LinkedIn posts and reflections.  
  - 🎓 **Skills & Certifications** — Interactive logo carousel and stacked certification cards.  
  - 🌍 **Life / Interests** — Anime art, fitness, and creative writing.  
  - 📬 **Contact** — Email, LinkedIn, GitHub, and resume download.

- **Responsive Design**  
  Optimized for desktop, tablet, and mobile.

- **Accessibility & SEO**  
  Semantic HTML, recruiter‑friendly keywords, Open Graph tags, and high contrast dark mode.

---

## 🛠 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)  
- **Styling**: [TailwindCSS](https://tailwindcss.com/) + shadcn UI  
- **Deployment**: [Vercel](https://vercel.com/) / Netlify  
- **Animations**: CSS transitions + Framer Motion  
- **Content**: Structured data in `src/data/siteData.ts`
